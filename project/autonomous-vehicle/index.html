Autonomous Robot

Multi-sensor autonomous robot built from scratch.

The Autonomous Robot was developed to navigate a multi-challenge obstacle course using real-time sensor input and state-driven control logic. It combines ultrasonic distance sensors, TCS230 color sensors, a gyroscope, and servo/DC motors to perform tasks such as line following, object avoidance, and directional turning.

Built around the Teensy 4.1 microcontroller and programmed in Arduino C++, the system features modular code for motor control, sensor integration, and navigation states. Custom CAD parts were 3D printed, and a tailored power distribution system was designed using buck converters. This project demonstrates full-stack embedded integration—from mechanical and electrical design to firmware architecture and testing.

Custom chassis, sensor mounts, and servo brackets designed in Autodesk Inventor and fabricated via 3D printing.
