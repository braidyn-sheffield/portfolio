Autonomous Robot

Multi-sensor autonomous robot built from scratch.

Overview

This robot navigates autonomously using multiple sensors and a state-machine controller. I designed and built the hardware, integrated the sensors, and wrote the firmware to handle navigation, obstacle detection, and decision-making in real time.

Highlights

  • Modular sensor stack (e.g., ultrasonic, color sensing, IMU) with debounced, fused readings.
  • State-machine architecture for predictable navigation and behavior transitions.
  • Stable power distribution for motors/servos and logic rails; clean signal routing.

Tech

  • MCU: Arduino/Teensy (C/C++)
  • Sensors: ultrasonic rangefinders, color sensors, MPU-6050 IMU
  • Actuation: DC motors + motor driver, servos
  • Firmware: non-blocking loops, timers, and sensor fusion basics
