Robotic Arm

3D-printed robotic arm with multi-servo control. Includes CAD, schematic, Arduino code, and final report.

Desktop robotic arm with 3D-printed links and mounts. Each joint uses hobby servos driven by an Arduino. Potentiometers provide manual inputs; code applies mapping, clamping, and smoothing for stable motion.

  • Joints: base, shoulder, elbow, wrist, gripper
  • Input: potentiometers → servo angles
  • Safety: end-stop limits and rate limiting
